# AI Tourist Surf Recommendation System

Building AI course project

## Summary

An AI-powered recommendation system that helps tourists find the best surf lessons, surf spots, and surfing schedules in Agadir, Morocco based on weather conditions, surf level, budget, and personal preferences.

---

## Background

Many tourists visiting surf destinations do not know:

* which surf spot matches their level
* the safest time to surf
* what equipment they need
* which lesson type is best for them

Beginners often choose unsuitable surf conditions, which can reduce safety and enjoyment.

As someone involved in surf coaching and tourism in Agadir, I noticed that many visitors struggle to make informed decisions about surfing activities. This project aims to improve the surf experience using AI recommendations based on real conditions and user preferences.

Surf tourism is growing worldwide, and AI can help create safer and more personalized surf experiences.

---

## How is it used?

The user enters information such as:

* surfing level
* preferred activity
* budget
* weather preferences
* lesson type

The AI system analyzes:

* wave conditions
* weather forecasts
* tide information
* user preferences

The system then recommends:

* suitable surf spots
* best lesson times
* recommended equipment
* beginner-friendly conditions

This solution could be used by:

* surf schools
* tourists
* beginner surfers
* travel agencies
* surf coaches

Example:

A beginner tourist visiting Agadir receives recommendations for calm beaches and safe surf times instead of advanced surf spots with dangerous waves.

## Data sources and AI methods

Possible data sources include:

* weather APIs

* tide and wave data

* surf school booking information

AI methods that could be used:

* recommendation systems
* machine learning classification
* predictive analytics
* natural language processing for chatbot support

The project could later integrate real-time weather and ocean data.

---

## Challenges

This project may not always predict ocean conditions perfectly because weather and waves can change quickly.

The quality of recommendations depends on the accuracy of external weather and surf data.

Ethical considerations include:

* protecting user data
* avoiding unsafe recommendations
* ensuring beginners receive safe guidance

AI should support human surf instructors, not replace professional judgment.
